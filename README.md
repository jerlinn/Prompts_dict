# Prompts 词典

**一份prompt快捷输入词典，收录了我在极客时间《零基础GPT应用入门课》的模版和prompt，Mac 全家桶通用。**

词典主要包含：
- 格式和示例
- 迭代话术
- 模版
- 人设
- 综合人设模版

## 使用

- 下载 ChatGPT.plist
- 打开 “系统设置 ▸ 键盘 ▸ 文本输入 ▸ 自定义短语“
- 拖入，🎉 搞定撒花。
